PHPStorm 2017.1.2 + Xdebug 远程调试

Posted on 2017-05-23 | In Tools

安装配置xdebug

XDebug官网： http://xdebug.org/

安装xdebug

如何选择xdebug的版本。

TS NTS

  • 如果是IIS系列，就选择带“Non Thread Safe”的版本
  • 如果是Apache httpd系列，就选择带“Thread Safe”的版本

vc6 vc9 vc11

说明PHP版本是由什么编译器进行编译的

  • VC6 ：Visual C++ 6
  • VC9 ：Visual C++ 2008
  • VC11 : Visual C++ 2012

修改PHP配置文件

phpstorm 设置

settings->Languages & Frameworks -> PHP -> Debug

Xdebug
Debug port : 改为上述对应的remote_port

settings->Languages & Frameworks -> PHP -> Debug -> DBGp Proxy

IDE key: 改为上述对应的 xdebug.idekey 的值
Port: 改为上述对应的remote_port

settings->Languages & Frameworks -> PHP -> Servers 设置对应的server即可
