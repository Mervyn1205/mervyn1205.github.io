嵌套

编译后的css

默认@at-root只会跳出选择器嵌套，不能跳出@media或@support，如果要跳出这两种，则需使用@at-root(without: media), @at-root(without:support).这个语法的关键词有四个 ：

all 表示所有
rule 表示常规css
media 表示media

默认的@at-root 其实就是@at-root(without: rule)

安装xdebug

如何选择xdebug的版本。

TS NTS

  • 如果是IIS系列，就选择带“Non Thread Safe”的版本
  • 如果是Apache httpd系列，就选择带“Thread Safe”的版本

vc6 vc9 vc11

说明PHP版本是由什么编译器进行编译的

  • VC6 ：Visual C++ 6
  • VC9 ：Visual C++ 2008
  • VC11 : Visual C++ 2012

修改PHP配置文件

phpstorm 设置

settings->Languages & Frameworks -> PHP -> Debug

Xdebug
Debug port : 改为上述对应的remote_port

settings->Languages & Frameworks -> PHP -> Debug -> DBGp Proxy

IDE key: 改为上述对应的 xdebug.idekey 的值
