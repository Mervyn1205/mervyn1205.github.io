Python3学习笔记（一）| 基本对象类型

发表于 2019-07-07 更新于 2019-07-12 分类于 Python

之前的文章已经讲过如何管理 Python 环境，所以这个入门系列直接略过安装的步骤。需要学习如何安装 Python、管理 Python 环境的朋友可以看之前的文章： 使用 pyenv 和 pyenv-virtualenv 管理python环境 。

在Python中，万物皆对象.比如整数3，就是一个对象。

内建函数 id() 查看每个对象的内存地址
内建函数 type() 能够查看对象的类型

数据类型

数值类型

整数

用 int 表示

Python可以处理任意大小的整数，不用担心溢出问题。同时也没有其他语言的长整数 这一类型。

浮点数

用 float 表示

一般笼统的把小数称之为“浮点数”。

浮点数跟整数不同，它存在上限和下限，如果超出了上下限的范围就会溢出。

四则运算

  • 除法运算的结果都是浮点数.
  • 特殊的 // 运算符，操作结果是取得两个数相除的商。 商 肯定是整数
  • 用 % 来取得两个数(可以是整数，也可以是浮点数)相除的余数

除了除法运算外，运算符两侧只要有一个是浮点数，结果就是浮点数，否则就是整数。

除了加减乘除以外，还有乘方、开方、对数等运算。为了便利地实现这些运算，可以使用math 模块。对于需要非常精确的情况，可以使用 decimal 模块.

ps: 单行注释用 # 开头，多行注释用一对 '''

字符串类型

python的字符串是一种对象类型，用 str 表示。通常用欧冠单引号或双引号包裹起来。

用 str() 可以将整数对象转换为字符串对象。

变量

对象有类型，变量无类型。
例：

5、6 都是整数，也叫做整数类型的对象。
变量 x、y 没有类型之说，因为它们既可以贴在整数类型的对象，也可以贴在其他类型的对象上。
